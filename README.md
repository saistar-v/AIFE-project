## ✨ Features

- **Dark Theme** — Elegant dark color scheme with purple accent gradients
- **Glassmorphism UI** — Modern frosted-glass card effects
- **Animated Background** — Floating gradient orbs with grid overlay
- **Typewriter Effect** — Auto-typing hero subtitle animation
- **Scroll Animations** — Smooth reveal-on-scroll for all sections
- **Interactive Cursor Glow** — Subtle glow that follows the mouse
- **Stats Counter** — Animated number counting on scroll
- **Responsive Design** — Fully responsive on all devices (mobile, tablet, desktop)
- **Mobile Navigation** — Hamburger menu with slide-in panel
- **Skill Progress Bars** — Animated skill level indicators
- **Contact Form** — Styled form with floating labels and visual feedback
- **SEO Optimized** — Proper meta tags, semantic HTML, heading hierarchy

## 🛠️ Getting Started

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/portfolio.git
   ```
2. Open `index.html` in your browser — no build tools or server required!

### Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings** → **Pages**.
3. Under **Source**, select **Deploy from a branch**.
4. Choose the `main` branch and `/ (root)` folder.
5. Click **Save** — your site will be live at `https://YOUR_USERNAME.github.io/portfolio/`

## 🎨 Customization

### Profile Photo
Replace `assets/profile.png` with your own photo. The image will display in the About section. If no image is provided, a stylish gradient placeholder with your initials is shown automatically.

### Content
Edit `index.html` to update:
- Personal information (name, age, email, location)
- Project descriptions and links
- Skills and technologies
- Social media links

### Colors
Edit the CSS variables at the top of `style.css`:
```css
:root {
    --accent-primary: #6c5ce7;     /* Main accent color */
    --accent-secondary: #a29bfe;   /* Secondary accent */
    --bg-primary: #0a0a0f;         /* Background color */
}
```
